1. What is Syncify?

Syncify is an innovative application that allows you to listen to music in a synchronized manner with your friends using Spotify APIs. You can create shared listening sessions and join existing ones, offering a collaborative and interactive musical experience. The application provides an intuitive interface for managing and controlling music playback, ensuring precise synchronization for a truly shared listening experience. Syncify is completely web-based, so you won't need to install anything! Just access the website, create a session or join an existing one, and start sharing music!

2. How does Syncify work?

Syncify utilizes the Spotify Web API to access and control music playback. When you create a session or join an existing one, Syncify establishes a connection between your Spotify account and the session.

3. How to install and use the Syncify plugin?

3.1. Installing the plugin

To install the Syncify plugin, you need to open your PowerShell as an administrator and run the following command:

3.2. Using the plugin

This command will download and run the installation script for the Syncify plugin. After the installation is complete, you can access the plugin by clicking on the Syncify icon in your Spotify application.

4. How to create or join a room

4.1. Creating a room

To create a room, simply click on the "Create Room" button on the homepage. You will be redirected to a new page where you can enter a room name and set the maximum number of participants. Once you've filled in the details, click on the "Create" button to create the room.

4.2. Joining a room

To join a room, click on the "Join Room" button on the homepage. You will be redirected to a new page where you can enter the room code. Click on the "Join" button to join the room.
